# Markdown Quick Start

## 基础语法

+ 1～6个#号代表不同级别的标题
+ +代表无序列表
+ 1.代表有序列表
+ \*\*代表加粗 **加粗**
+ \*代表斜体 *斜体*
+ \~\~代表删除 ~~删除~~
+ [我的博客](http://blog.yangdy.com)
+ ![我的头像](http://blog.yangdy.com/styles/images/me/me.png)
+ Table
